# GrowthIQ: AI-Powered Business Intelligence Engine

### *Predictive Analytics & Strategic Simulation for E-commerce*

## Overview
**GrowthIQ** is a Prescriptive Analytics dashboard designed to help e-commerce founders move from "guessing" to "data-driven decisions." 

Unlike standard dashboards that only show historical data, GrowthIQ uses **Facebook Prophet** (Machine Learning) to forecast future revenue and includes a **Strategy Simulator** that calculates the financial impact of business decisions (e.g., "How will a 10% discount affect my net profit?").

## Key Features
* ** AI Forecasting:** Predicts 30-90 day revenue trends using additive regression models.
* ** Strategy Simulator:** Simulates Price Elasticity of Demand to recommend optimal discount strategies.
* ** Global Scaling:** Instant currency conversion (USD/INR) for international reporting.
* ** Seasonality Detection:** Identifies "Best Days to Sell" automatically.

##  Tech Stack
* **Core Logic:** Python 3.10+
* **Machine Learning:** Facebook Prophet
* **Dashboard:** Streamlit
* **Visualization:** Plotly Interactive Charts

##  How to Run Locally
1.  **Clone the repository**
    ```bash
    git clone [https://github.com/YOUR_USERNAME/GrowthIQ-Sales-Forecaster.git](https://github.com/YOUR_USERNAME/GrowthIQ-Sales-Forecaster.git)
    cd GrowthIQ-Sales-Forecaster
    ```

2.  **Install Dependencies**
    ```bash
    pip install -r requirements.txt
    ```

3.  **Train the Model**
    ```bash
    python train_model.py
    ```

4.  **Run the Dashboard**
    ```bash
    streamlit run app.py
    ```
